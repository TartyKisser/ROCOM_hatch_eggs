## [v2.0.0] - 2026-06-02

> 版本高光：项目从单一“家园孵蛋排布工具”升级为完整的“洛克王国世界工具站”。

### 新增

- 新增网站主页，集中展示家园小窝排布、精灵图鉴、奇遇混抓推荐三个核心模块。
- 新增远行商人提醒栏目，可在主页标题下方展示当前时段在售商品。
- 新增 [`merchant_proxy.php`](./merchant_proxy.php)，在支持 PHP 的部署环境中为远行商人数据同步提供代理接口。
- 新增远行商人展示资源接入：商品背景框、商品图标背景、商人立绘、金币图标和 `web_images/shop/` 商品图目录。
- 新增主页“打赏”按钮，下拉展示两张打赏图片。
- 新增主页“反馈建议”按钮，用户可通过邮件客户端快速向 `tartykisser1022@gmail.com` 发送建议。
- 新增 [`rare.csv`](./rare.csv)，将“还可能触发”的稀有异色目标从奇遇目标数据中独立出来。
- 新增 [`evolution_chains.json`](./evolution_chains.json)，并将进化链信息合并进 [`handbook_data.json`](./handbook_data.json)。
- 新增精灵图鉴详情页的进化链展示区域，支持查看同一家族不同形态的图片和名称。
- 新增奇遇混抓推荐的“推荐规则说明”弹窗，内容来自 [`adventure_rules.txt`](./adventure_rules.txt)。
- 新增各模块页面右侧“回到顶部”按钮，主页除外。

### 调整

- 主页顶部信息区重新组织，将数据来源、WIKI、GitHub 和打赏入口整合在更紧凑的位置。
- 远行商人栏目改为更接近游戏内资源风格的黄色背景，并优化商人立绘与总框的层级关系。
- 远行商人价格展示改为使用 `coin.png`，价格数字显示在金币右侧。
- 远行商人商品名称与价格改为同一行展示，价格靠近商品框右侧。
- 移动端优化远行商人商人立绘位置，避免图片右侧超出总框。
- 精灵图鉴详情页中，精灵果实不再独占整行，改为左侧果实、右侧进化链的双栏结构。
- 精灵图鉴返回列表时保留原滚动位置，减少浏览长列表时的重复操作。
- 奇遇混抓推荐搜索区域重构为上下两块：
  - 上方选择奇遇目标家族；
  - 下方可选输入两种候选混抓家族。
- 奇遇目标、候选混抓对象和“还可能触发”均改为按家族维度展示。
- 奇遇混抓推荐排序方式从下拉框改为左右二选一按钮：
  - `星光值优先`
  - `少歪异色优先`
- 奇遇混抓推荐移除了“奇遇目标 / 候选混抓家族 / 可触发组合 / 当前策略”统计行，让结果区更清爽。
- 推荐规则说明改为面向玩家的表达，不再暴露内部数据文件和网站设计细节。

### 算法

- 奇遇混抓推荐按进化链合并家族，同一家族不同形态不再重复作为独立目标干扰推荐。
- 当一个精灵编号存在多条记录时，推荐计算只采用该编号的第一条记录。
- 家族星光值计算调整为加权平均：
  - 单形态：使用自身星光值；
  - 双形态：`(小形态 * 3.5 + 大形态 * 1.5) / 5`；
  - 三形态：`(小形态 * 2.5 + 中形态 * 1.5 + 大形态 * 1) / 5`。
- 混抓组合的星光值之和改为两个候选家族加权星光值相加。
- `星光值优先` 排序先看星光值之和，再看“还可能触发”的家族数量。
- `少歪异色优先` 排序先看“还可能触发”的家族数量，再看星光值之和。

### 修复

- 修复远行商人数据在浏览器端直接请求时容易因跨域限制导致“数据抓取失败”的问题，改为优先使用 PHP 代理。
- 修复远行商人移动端商人图片最右侧超出总框的问题。
- 修复奇遇混抓推荐中，选择候选混抓家族后仍提示“没有匹配的混抓策略”的问题。
- 修复奇遇家族显示中多余冒号的问题。
- 移除部分强制显示家族的临时逻辑，改为依赖已恢复的进化链数据。
- 移除网站对反馈内容的本地保存逻辑，不再使用 `feedback_submit.php` 和 `feedback_submissions/`。
- 修复精灵图鉴从详情返回后自动回到顶部的问题。

### 数据

- 更新 `handbook_data.json`，加入 `evolution` 字段。
- 保留 `evolution_chains.json` 作为进化链源数据备查。
- 将奇遇推荐规则整理为 `adventure_rules.txt`，便于页面渲染与后续维护。
- 新增和整理远行商人商品图片目录 `web_images/shop/`。
