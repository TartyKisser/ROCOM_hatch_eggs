## 这个网站能做什么

这个工具站主要解决几件常见的小麻烦：

- 家园小窝不知道怎么摆，想尽量多配几对。
- 想快速查某只精灵的属性、蛋组、技能、果实收益和进化链。
- 奇遇混抓时，不确定哪两只精灵更适合一起抓。
- 想知道远行商人现在卖什么，不想反复切页面确认。

网站会尽量把复杂信息整理成可以直接看的结果。你只需要选目标、输入精灵，剩下的交给工具计算和展示。

## 功能介绍

### 家园小窝排布

适合想在家园里安排多只精灵孵蛋的玩家。

- 支持选择小窝数量。
- 输入精灵名称时会自动提示。
- 可以设置精灵性别。
- 自动计算更合适的摆放方案。
- 展示小窝位置、精灵头像和可配对明细。

简单来说：把你准备放进小窝的精灵填进去，网站会给你一份更容易照着摆的方案。

### 精灵图鉴

适合查精灵资料、对比培养信息、确认进化关系。

- 支持按编号、名称等信息搜索。
- 可以查看精灵属性、蛋组、技能、特性等资料。
- 精灵详情页会展示果实相关信息。
- 有进化链的精灵会展示同一家族形态。
- 没有进化链的精灵也会明确提示。

进入详情后再返回列表时，页面会尽量回到你刚才浏览的位置，减少重复翻找。

### 奇遇混抓推荐

适合想更高效规划奇遇异色混抓的玩家。

- 可以先选择想要的奇遇目标。
- 可以指定一到两种想参与混抓的精灵家族。
- 输入部分名称或编号时，会弹出候选项。
- 推荐结果会展示可用组合、星光值表现，以及可能额外触发的目标。
- 支持两种排序：
  - **星光值优先**：更看重组合整体星光值。
  - **少歪异色优先**：更看重减少额外触发目标。

页面里也提供了“推荐规则说明”，想看得更细时可以直接点开。

### 远行商人提醒

主页会展示当前远行商人的售卖信息。

- 可以看到当前时段在售商品。
- 商品会尽量配上对应图标和价格。
- 如果同步失败，可能是来源页面、网络环境或服务器限制导致，稍后刷新即可。

## 使用建议

1. 第一次进入可以先从主页选择需要的工具。
2. 搜索精灵时，输入编号或名字的一部分即可。
3. 奇遇混抓推荐不一定只有一个正确答案，可以根据你当前拥有的精灵和目标灵活选择。
4. 如果发现资料有误、商品缺图或推荐结果不符合预期，欢迎通过主页的反馈按钮联系我。

## 常见问题

**Q：为什么有些图片没有显示？**  
A：通常是对应图片暂时还没有补齐，后续会继续更新。

**Q：为什么远行商人偶尔同步失败？**  
A：商人信息需要从外部页面同步，遇到网络波动、来源页面变化或服务器限制时可能失败。可以稍后刷新再试。

**Q：奇遇混抓推荐一定是最优解吗？**  
A：推荐结果是根据当前整理出的资料和规则进行计算，适合用作规划参考。实际选择时还可以结合你自己的精灵库存、抓捕习惯和当前目标。

**Q：发现资料错误怎么办？**  
A：可以在主页点击“反馈建议”，把问题、精灵名称和你看到的情况发给我。
